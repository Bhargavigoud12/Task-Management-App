**How to Run**
**Clone this repo**
git clone https://github.com/your-username/task-management-app.git

Install dependencies for both frontend & backend
cd frontend
npm install

cd ../backend
npm install

Start the backend server
npm start

Start the frontend app
cd frontend
npm start
Open in browser
